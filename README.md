# Gestion des Factures
Application web pour gérer les clients et les factures, incluant la génération de PDF et un tableau de bord interactif.

## Fonctionnalités
- Ajouter, modifier et supprimer des clients
- Créer, modifier et supprimer des factures
- Générer des factures au format PDF
- Tableau de bord avec statistiques
- Gestion des rôles : Administrateur et Utilisateur

## Technologies
- Frontend : React.js, HTML, CSS
- Backend : PHP
- Base de données : MySQL 
- Authentification : JWT
- Génération de PDF : FPDF
- Graphiques et statistiques : Chart.js
